# Bibliotecas do Python

## O que é uma biblioteca?
- Conjunto de módulos e funções que podem ser utilizados para ampliar o uso da linguagem
- As bibliotecas são pensadas para facilitar a criação de programas, fornecendo um conjunto de ferramentas e recursos prontos para uso 

## Como usá-las
- As bibliotecas em python são, geralmente, distribuídas como pacotes, que podem ser instalados por meio de gerenciadores de pacotes, como o pip

```python
# 5 - Agora vamos preparar alguns ndarrays para serem representados por gráficos:  
vetor_a = np.linspace(10,1000,100)  
vetor_b = np.linspace(10,3000,100)  
vetor_c = np.linspace(10,9000,100)  
  
print(vetor_a)  
print(vetor_b)  
print(vetor_c)  
  
# Numpy já tem métodos bem diretos de como salvar um arquivo no formato .txt  
np.savetxt('vetor_a.txt', vetor_a, fmt = '%f', delimiter=';')  
np.savetxt('vetor_b.txt', vetor_b, fmt = '%f', delimiter=';')  
np.savetxt('vetor_c.txt', vetor_c, fmt = '%f', delimiter=';')
```


